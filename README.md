# C# OOP Masterclass

Bu proje, C# programlama dilinde Nesne Yönelimli Programlama (OOP) kavramlarını detaylı olarak anlatan, modern ve interaktif bir eğitim uygulamasıdır. İçerikte, C# OOP temellerinden ileri düzey konulara kadar birçok konu ele alınmaktadır.

## Özellikler

*   Kapsamlı C# OOP eğitim içeriği.
*   Kod örnekleri ve açıklamalar.
*   Modern, duyarlı (responsive) React tabanlı kullanıcı arayüzü.

## Teknolojiler

*   React
*   Vite
*   Tailwind CSS

## Kurulum ve Çalıştırma

Projeyi yerel bilgisayarınızda çalıştırmak için:

1.  Bağımlılıkları yükleyin:
    ```bash
    npm install
    ```
2.  Geliştirme sunucusunu başlatın:
    ```bash
    npm run dev
